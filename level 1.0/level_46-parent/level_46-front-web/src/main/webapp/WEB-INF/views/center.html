常见问题
  1. Q1：忘记准考证号怎么办？

    点击这里查询。

  2. Q2：为什么笔试成绩为0？

    笔试成绩为0有以下几种情况：

    （1）缺考违规成绩为0分；

    （2）未缺考违规但成绩在220分以下则报道为0分；

    （3）未缺考违规但在考试过程中未粘贴条形码，则成绩为0分。

  3. Q3：如对成绩有异议怎么办？

    如对成绩有异议，可按规定进行分数核查，具体办法请见成绩核查。
